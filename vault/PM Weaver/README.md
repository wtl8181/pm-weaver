# PM Weaver Vault

This folder is the project-local Obsidian-compatible vault used by PM Weaver.

Runtime files are written here by the local API:

- `Tasks/*.md`: one Markdown file per task.
- `Workflows/{taskId}.workflow.json`: task-scoped React Flow nodes, edges, node status, input snapshots, and outputs.
- `Artifacts/*.md`: message and PRD Markdown artifacts with frontmatter metadata.

You can open `vault/PM Weaver` directly as an Obsidian vault.
